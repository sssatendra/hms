# HMS Enhancements Guide

This guide provides an overview of the new features integrated into the Hospital Management System (HMS) and how to use them effectively.

## 1. 🔍 Global "Command-K" Search
The Global Search is a centralized hub for navigating the system and finding records instantly.

- **How to Use**: Press `Ctrl + K` (Windows/Linux) or `Cmd + K` (Mac) from any screen.
- **Searchable Entities**:
  - **Patients**: Search by Name, MRN, or Phone.
  - **Pharmacy/Inventory**: Search for medicines, stock levels, and HSN codes.
  - **Navigation**: Jump directly to Wards, Lab, Settings, etc.
- **Shortcut Keys**:
  - `P` -> Patients
  - `I` -> Inventory
  - `L` -> Lab

## 2. 📄 Professional PDF Reports
Standardized, hospital-branded PDF generation for clinical and administrative needs.

- **Available Reports**:
  - Lab Test Results (from Lab module).
  - Patient Discharge Summaries (from Wards module).
  - Invoices & Billing (from Billing module).
- **Customization**: The hospital logo and name are automatically pulled from the **Settings** page.

## 3. 实时 (Real-time) Activity Stream
Stay updated with live events without refreshing the page.

- **Live Notifications**:
  - **Low Stock Alerts**: Instant notification when a medicine falls below its reorder level.
  - **Lab Completions**: Alerts when a lab technician uploads a result.
- **Integrated Feedback**: Uses subtle toast notifications in the corner of the screen.

## 4. 🌙 Dark Mode Support
A high-fidelity dark theme designed for night-shift doctors and nurses.

- **Toggle**: Access the theme switch in the **Settings > Preferences** or the top navigation bar.
- **Benefit**: Reduces eye strain and preserves battery on mobile tablets.

## 5. 📉 Advanced Inventory Forecasting
Data-driven insights to prevent stock-outs.

- **Forecasting Tab**: Located within the **Inventory** and **Pharmacy** pages.
- **Days Remaining**: Shows an estimate of how many days current stock will last based on historical usage.
- **Reorder Alerts**: Highlights items that need urgent procurement.

## 6. 🌐 Internationalization (i18n)
Multi-language support for diverse hospital environments.

- **Supported Languages**: English (Default), with support for adding localized dialects and medical terminology.
- **Setting**: Change the language in **Settings > Profile**.
